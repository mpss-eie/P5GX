Módulo de dimensionamiento
==========================

Este módulo permite el diseño tanto del número de servidores como del tiempo promedio de servicio necesarios para satisfacer ciertos criterios de servicio.

.. important::
   El número de servidores :math:`s` y el parámetro de tiempo de servicio :math:`\nu` pueden ser ajustados individualmente o en conjunto.
